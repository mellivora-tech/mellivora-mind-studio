# 系统架构文档

## 一、系统概述

私募基金投研报告系统，支持 LP/投资人查看持仓和报告，买方进行投研分析和交易分析。

---

## 二、服务划分

| 服务 | 职责 |
|-----|------|
| C 端 Web | LP/投资人、买方业务界面 |
| C 端 APP | LP 移动端（iOS/Android） |
| C 端小程序 | LP 微信小程序 |
| 管理后台 | 运维人员管理界面 |
| API 网关 | 统一入口、路由、限流 |
| Agent 服务 | AI 对话、报告生成、智能问答 |
| 业务 API | 基金、报告等业务数据 |
| 用户服务 | 用户管理、角色、权限 |
| 认证服务 | 登录、Token、授权 |
| 数据库 | 业务数据存储 |
| 缓存 | 会话、热数据 |
| 文件存储 | 报告 PDF、图片 |

---

## 技术栈

### 前端

| 服务 | 技术 |
|-----|------|
| C 端 Web | SolidJS + Vite |
| C 端 APP | React Native |
| C 端小程序 | 原生微信小程序 |
| 管理后台 | React + Ant Design Pro |

### 后端

| 服务 | 技术 |
|-----|------|
| API 网关 | Hono |
| Agent 服务 | VoltAgent + Bun |
| 业务 API | Hono + Bun |
| 用户/认证 | Hono + Bun |

### 基础设施

| 服务 | 技术 |
|-----|------|
| 数据库 | Neon (PostgreSQL) |
| 缓存 | Upstash (Redis) |
| 文件存储 | Cloudflare R2 |

### 部署

| 服务 | 平台 |
|-----|------|
| 前端 | Vercel |
| 后端 | Railway |
| 数据库 | Neon |
| 缓存 | Upstash |

## 四、部署模式

### 免费版（公有云）

| 特点 | 说明 |
|-----|------|
| 部署 | 公有云托管 |
| 租户 | 多租户共享 |
| 数据 | 租户 ID 隔离 |
| 功能 | LP 看板、报告下载、通用问答 |

### 付费版（私有化 Provider）

| 服务 | 部署位置 | 说明 |
|-----|---------|------|
| 前端 | 平台提供 | Web/APP/小程序 |
| Agent 服务 | 平台提供 | AI 对话、报告生成 |
| 认证服务 | 平台提供 | 登录、License 验证 |
| 业务数据 API | **客户私有化** | 净值、交易、持仓等 |
| 数据库 | **客户私有化** | 敏感数据本地存储 |

### 私有化 Provider 架构

```
┌─────────────────────────────────────────────────────┐
│                  平台服务（公有云）                   │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐               │
│  │  前端   │ │ Agent   │ │  认证   │               │
│  │Web/APP │ │  服务   │ │  服务   │               │
│  └────┬────┘ └────┬────┘ └─────────┘               │
│       │           │                                 │
└───────┼───────────┼─────────────────────────────────┘
        │           │
        │           │ 调用私有化数据API
        │           ▼
┌───────┼─────────────────────────────────────────────┐
│       │       客户私有化部署                         │
│       │  ┌───────────────────────────────────┐     │
│       └─▶│ 业务数据 API（净值/交易/持仓）     │     │
│          └───────────────┬───────────────────┘     │
│                          ▼                          │
│                    ┌─────────┐                      │
│                    │ 数据库  │                      │
│                    └─────────┘                      │
│               (客户自己的服务器)                     │
└─────────────────────────────────────────────────────┘
```
