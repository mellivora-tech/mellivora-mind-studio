## 一、LP/投资人 功能

| 功能 | 描述 |
|-----|------|
| 我的基金 | 查看持有基金、份额 |
| 收益查看 | 净值、收益率、历史业绩 |
| 风险指标 | 回撤、波动率、夏普比率 |
| 报告下载 | 月报、季报、年报 |
| 账户明细 | 资金进出、分红记录 |

## 二、买方功能

| 功能 | 描述 |
|-----|------|
| 报告生成 | 生成投研报告 |
| 舆情监控 | 新闻、舆情分析 |
| 交易分析 | 见下方详细功能 |

---

## 三、AI 助手（双模式）

| 模式 | 描述 |
|-----|------|
| 通用问答 | 开放式问答，无特定上下文 |
| 深度研究 | 选择标的后，围绕该标的深入对话 |

**深度研究模式功能：**

| 功能 | 描述 |
|-----|------|
| 标的选择 | 搜索/从持仓/从自选选择标的 |
| 自动摘要 | 选择后自动生成标的总结 |
| 深度对话 | 基于标的上下文的问答 |
| 研究笔记 | 保存重要结论 |

---

## 三、交易分析

| 功能 | 描述 |
|-----|------|
| 行情看板 | K线、分时、技术指标 |
| 基准对比 | 与沪深300等基准对比 |
| 市场概览 | 板块热力图、涨跌幅排行 |
| 因子暴露分析 | 组合在各因子上的暴露度 |
| 因子收益归因 | 收益来源于哪些因子 |
| 风格漂移监控 | 策略风格是否发生变化 |
| 执行效率分析 | 意图单 vs 实际成交对比 |
| 执行缺口分析 | 下单到成交的价差 |
| 交易成本分析 | 佣金、印花税、滑点 |
| 冲击成本分析 | 大单对市场的影响 |
| TWAP/VWAP 对比 | 算法执行效果评估 |
| 持仓结构分析 | 行业/个股分布 |
| 集中度分析 | 前N大持仓占比 |
| 持仓变动追踪 | 加减仓记录 |
| 账户盈亏分析 | 已实现/未实现盈亏 |
| 资金使用率 | 仓位水平监控 |
| 收益曲线 | 日/周/月收益走势 |
| 最大回撤监控 | 历史最大回撤、当前回撤 |
| 回撤预警 | 触发阈值告警 |
| 水下时间分析 | 回撤持续周期 |
| 策略回测 | 历史业绩模拟 |
| Alpha 分析 | 超额收益来源 |
| 风险模型 | VaR、波动率预测 |
